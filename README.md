
# Sentiment Analysis of Product Reviews

## Description
This project performs sentiment analysis on product reviews using Natural Language Processing (NLP) and Machine Learning techniques. The reviews are classified as Positive, Negative, or Neutral.

## Technologies Used
- Python  
- NLP (NLTK)  
- Scikit-learn  
- Flask  
- HTML, CSS  

## Dataset
Product reviews dataset in CSV format containing review text and sentiment labels.

## How It Works
1. Text preprocessing using NLP techniques  
2. Feature extraction using TF-IDF  
3. Training a machine learning model  
4. Predicting sentiment of user input  

## How to Run
1. Install required libraries  

pip install -r requirements.txt

2. Run the application  

python app.py

3. Open browser and visit  

http://127.0.0.1:5000/

## Output
The system predicts whether a review is Positive, Negative, or Neutral.
